## About

This repository hosts the interactive web version of *The Hierarchical Universe*, a self-contained document that:

- Builds an infinite branching tree from a single primitive act (drawing a line)
- Defines a new way of measuring distance based on shared ancestry
- Shows how numbers, programs, and quantum states all live in the same tree geometry
- Explains why primes look random and program termination is unpredictable
- Proposes a **grand correspondence** — a precise dictionary translating between arithmetic, computation, physics, and semiotics
- Provides a blueprint for constructing hierarchical quantum computers with passive error immunity

## Local Development

To run locally:

```bash
python -m http.server 8000
# or
npx http-server
```

Then open `http://localhost:8000` in your browser.
